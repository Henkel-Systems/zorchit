# ⚡ Zorchit

> **Don't overcomplicate your task queues. Just Zorchit.**

Zorchit is a blazingly fast, highly reliable task orchestrator and job queue. By combining the rock-solid transactional state of **PostgreSQL** with the raw performance and safety of **Zig**, Zorchit delivers massive throughput in a single, lightweight binary.

## Why Zorchit?

* **⚡ Lightning Fast:** Written in Zig, meaning no garbage collection pauses, tiny memory footprint, and native performance.
* **🐘 Rock-Solid State:** Uses PostgreSQL as the source of truth. If your database survives, your jobs survive.
* **📦 Single Binary:** No massive runtimes or complex dependencies to install. Just drop the controller executable on your server and drop the agent on your hosts pointed to the controller.
